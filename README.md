# SPY Direction Prediction

## Overview
This project applies machine learning models to predict the next-day direction of SPY using financial market data and macroeconomic features.

## Objective
The goal of this project is to classify whether SPY will move up or down on the next trading day.

## Methods
The following machine learning models were implemented and compared:
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- XGBoost

## Data
The project uses market-related data and additional macroeconomic information, including the federal funds rate (`FEDFUNDS.csv`).

## Results
Among the tested models, SVM achieved the best overall performance in predicting next-day SPY direction.
